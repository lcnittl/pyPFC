This packages can be used to control an Argon Fan HAT.

## Usage

### Installation

Installable with `pip` or [`pipx`][pipx] (recommended).

```cmd
pipx install git+https://github.com/lcnittl/DMFO.git
```

**Note:** Adding [PyPI][pypi] support is planned.
